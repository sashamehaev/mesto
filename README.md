# Проект: Место

**Функциональность**
* Проект Mesto - это социальная сеть для фотографов со всей России.
* Вы можете загружать свои фотографии, смотреть фотографии других пользователей, ставить им лайки, а также редактировать свой профиль.

**Стек технологий**
* ООП
* vanilla js
* Javascript
* API
* Webpack
* Адаптивная верстка
* БЭМ
* HTML
* CSS
